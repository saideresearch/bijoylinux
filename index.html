Manual Configuration for Arch Linux

Step 1: Install Prerequisites

sudo pacman -S ibus-m17n m17n-db m17n-lib

Step 2: Get Configuration Files

git clone https://github.com/saideresearch/bijoylinux.git
cd bijoyLinux

Step 3: Install Mapping Files

sudo cp m17n.d/bn-bijoyUnicode.mim /usr/share/m17n/
sudo cp m17n.d/bn-bijoyClassic.mim /usr/share/m17n/

Step 4: Configure IBus

ibus restart
  • Open IBus Preferences: ibus-setup
  • Go to Input Method tab
  • Click Add → select Bengali - Bijoy Bayanno
  • Remove any other Bengali input methods
  • Set Bijoy Bayanno as default

Step 5: Test

Open any text editor and start typing with Bijoy Bayanno layout.

Source: github.com/nazdridoy/bijoyLinux

Windows

Download bijoy52-windows.zip (4.9 MB)

Installation Instructions

1. Extract the zip file:

  • Right-click bijoy52-windows.zip
  • Select Extract All

2. Run the installer:

  • Open the extracted folder
  • Right-click setup.exe
  • Select Run as Administrator

3. Follow installation wizard:

  • Click Next → Next → Install
  • Restart your computer when prompted

4. Activate Bijoy Bayanno:

Press Ctrl + Shift + B
